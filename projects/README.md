# Personal External Projects Hub

This directory holds the "brain" context for personal projects whose code resides in external repositories.

## Purpose
Instead of keeping a `todo.txt` in those repos, you can store your architectural decisions, task lists, and long-term goals here. Your AI agents will read these markdown files to help you strategize.

## Structure
Create a subfolder for each project (e.g., `langfuse/`, `ai-repo/`) and add a `log.md` or `architecture.md`.
