- **Non-Dev Teams:** Automating PR reviews, UAT deployments, release notes, and involving POs/DMs via Jira integration.
- **Tools to Evaluate:** GitHub Agentic Workflows, Copilot Spaces, AI Code Review tools, and MCP servers.

### 3. AI Transition Steps Plan (`steps-plan.md`)
A structured maturity model proposing how to transition teams to AI-driven development:
- **Level 1:** Enable Code Reviews for Each Repo
- **Level 2:** Setup GitHub Copilot for All Developers
- **Level 3:** Create Repository Instructions (`.github/copilot-instructions.md`)
- **Level 4:** Analyze Requirements using AI
- **Level 5:** AI-Assisted Implementation Planning
- **Level 6:** Utilizing GitHub Copilot Plan Mode
- **Level 7:** Establishing a Custom Prompts Library
