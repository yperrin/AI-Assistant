### **The Core Strategy: Systems Over Skills**

### **The 4 Layers of an Agentic OS**

1. **Shared Brand Context:** Instead of generic outputs, skills point to a central folder containing your brand voice, Ideal Customer Profile (ICP), and samples. This ensures every output sounds like you. [02:43 Opens in a new window](http://www.youtube.com/watch?v=5AfSB0sWihw&t=163)

1. **Continuous Learning Loop:** The system includes a `learnings.md` file. It logs your feedback (e.g., "this newsletter was too long") and automatically updates the `skill.md` instructions so you never have to repeat yourself. [06:22 Opens in a new window](http://www.youtube.com/watch?v=5AfSB0sWihw&t=382)

1. **Self-Maintenance (The "Heartbeat"):** Every time a session starts, the system scans for new folders or MCP servers. It automatically updates documentation and ensures new skills don't overlap with existing ones. [07:43 Opens in a new window](http://www.youtube.com/watch?v=5AfSB0sWihw&t=463)

1. **Chained Workflows:** Skills are categorized (Strategy, Execution, Ops) and linked. For example, a **Trending Research** skill finds topics on Reddit, which then feeds into a **Content Repurposing** skill to create a newsletter. [10:33 Opens in a new window](http://www.youtube.com/watch?v=5AfSB0sWihw&t=633)

### **Key Technical Components**

- `**skill.md**`: Step-by-step process documentation for a specific task. [01:26 Opens in a new window](http://www.youtube.com/watch?v=5AfSB0sWihw&t=86)

- **Deep Knowledge Files**: The references, assets, and scripts that support the instructions. [01:39 Opens in a new window](http://www.youtube.com/watch?v=5AfSB0sWihw&t=99)

- **Context Files**: Includes `soul.md` (agent identity), `user.md` (your preferences), and `memory.md` (long-term knowledge). [05:32 Opens in a new window](http://www.youtube.com/watch?v=5AfSB0sWihw&t=332)
