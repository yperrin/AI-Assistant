### **Key Market Moves**

- **Nvidia & NemoClaw:** Jensen Huang declared that every software company needs an "OpenClaw strategy." Nvidia introduced **NemoClaw**, an enterprise-grade toolkit that adds privacy, security, and isolated sandboxing to OpenClaw instances [07:43 Opens in a new window ](http://www.youtube.com/watch?v=zjcjT6BukQA&t=463).

- **OpenAI's "Code Red" Refocus:** OpenAI is reportedly abandoning "side quests" (like specialized hardware or niche apps) to focus strictly on **enterprise productivity and coding**. This includes new **sub-agent** capabilities in Codex, allowing users to spin up specialized agents for parallel tasks [12:18 Opens in a new window ](http://www.youtube.com/watch?v=zjcjT6BukQA&t=738).

- **Perplexity Computer:** A total reimagining of their UI into a "problem-solution design system" that can manage complex workflows across 400+ integrated applications [03:01 Opens in a new window ](http://www.youtube.com/watch?v=zjcjT6BukQA&t=181).

- **Meta & Manus:** Following its acquisition by Meta, **Manus** launched a desktop app ("My Computer") designed to perform local tasks like organizing thousands of files or building Swift apps without manual code [04:13 Opens in a new window ](http://www.youtube.com/watch?v=zjcjT6BukQA&t=253).

### **The "Local-First" Trend**

- **Adaptive Computer:** Features "encoded memory" that learns how you use specific software (like Square or Excel) and automates repetitive business workflows locally [05:43 Opens in a new window ](http://www.youtube.com/watch?v=zjcjT6BukQA&t=343).

- **Integration:** Builders are converging on the idea that for an agent to be truly useful, it must have access to your local file system, development environment, and essential desktop applications [05:09 Opens in a new window ](http://www.youtube.com/watch?v=zjcjT6BukQA&t=309).
