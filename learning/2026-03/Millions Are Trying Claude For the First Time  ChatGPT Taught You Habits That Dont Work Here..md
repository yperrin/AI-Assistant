### **Core Differences in Philosophy**

- **Training Method:** ChatGPT uses Reinforcement Learning from Human Feedback (RLHF), which optimizes for user satisfaction and agreeableness [03:57 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=237). Claude uses **Constitutional AI**, training the model against explicit principles like honesty and helpfulness, making it more likely to prioritize correctness over pleasing the user [02:28 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=148).

- **Behavioral Default:** ChatGPT tends to be "sycophantic" (telling you what you want to hear) [03:09 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=189), whereas Claude is more likely to flag holes in your logic or question your framing [04:14 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=254).

### **The 7 Principles for Using Claude**

1. **Accept Pushback:** Claude is designed to stress-test your plans. It is more likely to challenge a flawed strategy than ChatGPT [04:52 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=292).

1. **Describe Situations, Not Just Outputs:** Instead of just giving commands (e.g., "write a letter"), describe the context. Claude excels at reasoning about the framing of a task [05:25 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=325).

1. **Use it as an Editor:** Claude is often better at refining existing work than generating from a blank canvas. It tends to have a more natural, less "AI-sounding" voice [07:34 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=454).

1. **Ask for Reasoning:** Claude uses "extended thinking" to work through complex problems step-by-step. You can monitor this chain of thought and intervene if it goes off track [10:04 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=604).

1. **Build Workspaces (Projects):** Use Claude Projects to set persistent "operating rules" for specific domains (e.g., marketing guidelines) rather than just using them as file storage [12:34 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=754).

1. **Utilize "Co-work":** Unlike ChatGPT, Claude has a desktop agent ("Co-work") that can autonomously open, read, and edit files on your computer [16:27 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=987).

1. **Know the Trade-offs:** By switching to Claude, you lose features like native image generation, Sora video, and the ChatGPT GPT Store [17:42 Opens in a new window ](http://www.youtube.com/watch?v=O7SSQfiPDXA&t=1062).
