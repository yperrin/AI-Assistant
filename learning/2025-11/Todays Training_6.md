# AI News of the week

- **Gemini 3 Launch:** Google released its new flagship "thinking" model with major improvements in reasoning, coding, and multimodal understanding. It is available now in the Gemini web app, API, and for some users in Google Search. [00:42 Opens in a new window](http://www.youtube.com/watch?v=GO5TngVcH1A&t=42)

- **Gemini Agent & Anti-Gravity:** A new agent mode can perform multi-step tasks (e.g., browsing, planning), and "Anti-Gravity" is a new AI-powered IDE for developers. [02:00 Opens in a new window](http://www.youtube.com/watch?v=GO5TngVcH1A&t=120)

- **Nano Banana Pro:** A new state-of-the-art image generation model built on Gemini 3. It excels at text rendering, blending up to 14 images, and style transfer. It is available in the Gemini app for users with "thinking mode" enabled. [03:50 Opens in a new window](http://www.youtube.com/watch?v=GO5TngVcH1A&t=230)

- **AI Agents in Windows:** Microsoft is integrating AI agents directly into the Windows 11 taskbar and File Explorer to automate tasks and summarize files. [11:20 Opens in a new window](http://www.youtube.com/watch?v=GO5TngVcH1A&t=680)

- **Office Integration:** New agents for Word, Excel, and PowerPoint can create documents from simple prompts.

- **Anthropic Partnership:** Microsoft, Nvidia, and Anthropic announced a partnership, with Microsoft bringing Claude models to the Copilot ecosystem. [10:33 Opens in a new window](http://www.youtube.com/watch?v=GO5TngVcH1A&t=633)

- **xAI Grok 4.1:** xAI released Grok 4.1, which briefly topped leaderboards before Gemini 3's release. It is noted for high emotional intelligence and low hallucination rates. [12:40 Opens in a new window](http://www.youtube.com/watch?v=GO5TngVcH1A&t=760)

- **Meta SAM 3 & SAM 3D:** Meta released "Segment Anything Model 3" for precise object tracking in video and "SAM 3D" for turning 2D images into 3D objects. [13:40 Opens in a new window](http://www.youtube.com/watch?v=GO5TngVcH1A&t=820)

- **OpenAI Updates:** OpenAI launched **GPT 5.1 Codex Max**, a massive context model for deep coding tasks. They also introduced a "Group Chat" feature for ChatGPT and a free version specifically for teachers. [18:29 Opens in a new window](http://www.youtube.com/watch?v=GO5TngVcH1A&t=1109)

- **Replet:** Added a new design feature powered by Gemini 3.0.

- **11 Labs:** Expanded beyond audio into image and video generation.

- **Manis:** Released a browser extension for agentic web browsing.

- **Midjourney:** Rolled out new user profiles with social features.
