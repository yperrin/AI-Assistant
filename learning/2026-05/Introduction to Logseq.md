### 1. Key Point Extraction

- **Privacy and Local Storage:** Logseq acts as a local-first interface that stores data directly on your computer in **Markdown** or **Org-mode** formats, ensuring you own your data [01:11 Opens in a new window ](http://www.youtube.com/watch?v=WtRubK0YoyU&t=71).

- **The "Graph" Concept:** Notes are organized into a "graph" rather than a traditional folder hierarchy, allowing for non-linear connections between thoughts [00:52 Opens in a new window ](http://www.youtube.com/watch?v=WtRubK0YoyU&t=52).

- **Bidirectional Linking:** Using double square brackets `[[ ]]` creates pages and links automatically. This "Personal Wiki" structure allows you to see every place a specific topic or person was mentioned [03:34 Opens in a new window ](http://www.youtube.com/watch?v=WtRubK0YoyU&t=214).

- **Daily Journaling:** The app defaults to a "Journal" view, encouraging users to capture thoughts chronologically, which can then be tagged or linked to specific project or topic pages [03:05 Opens in a new window ](http://www.youtube.com/watch?v=WtRubK0YoyU&t=185).

- **Task Management:** It includes built-in workflow features like `TODO`, `DOING`, and `DONE` statuses, which can be managed using powerful internal queries to create custom dashboards [05:31 Opens in a new window ](http://www.youtube.com/watch?v=WtRubK0YoyU&t=331).

- **Visualizing Knowledge:** The "Graph View" provides a visual map of how different notes and topics are interconnected, helping users discover relationships serendipitously [05:11 Opens in a new window ](http://www.youtube.com/watch?v=WtRubK0YoyU&t=311).
