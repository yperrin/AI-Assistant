# The Research Phase

- Define the Goal

	- Use AI to define the boundaries on the topic (any of them, use advanced + internet search)

		- what should I learn

		- what tools are currently available

		- which ones have the more educational value

- Initial Research + information gathering to create a personalized learning roadmap

	- Use Perplexity to find resources 

	- Add the resources into NotebookLM - also use the advanced search capabilities

- Controversy search

	- Why does it fail?

	- Compare pros and cons with similar approaches or tools

- Verify sources

	- Identify any areas my sources disagree on and any contradictions between them

	- Identify gaps in my sources

	- Are there any contrarian or less known view points not covered in my sources?

- Build Knowledge foundation

	- YouTube playlists

	- Web Articles

	- Create notes Notion/Obsidian/Google docs but only if you create new thoughts, any other notes can be included in NotebookLM and potentially added as sources.

# The Plan Phase

- Do guided hands on learning

	- Use NotebookLM tools podcast, mind map, visual presentation to get started using the medium which works the best for you.

	- Combine with Gemini/ChatGPT/Claude to create graphics, infographics and interactive web pages to deepen you understanding.

- Reinforce and test understanding

	- As NotebookLM to explain a concept and then ask you to explain it and critic your answers

# The Build Phase

- Work on projects

	- Create a new project, hopefully a small scale, to make sure you validate your understanding.

	- Ask AI to create a project for you based on your preferences and current level.

	- When coding you can give Agent instructions to be an AI Mentor and generate explanations as well as code

	- For the code you wrote, as AI to review and provide guidance.

- Strengthen understanding by teaching

	- Create a team’s post

	- Teach your colleagues
