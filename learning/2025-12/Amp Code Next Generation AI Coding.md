In this video, **Beyang Liu** introduces **Amp Code**, an "opinionated frontier agent" for coding that focuses on high-level orchestration and sub-agent architecture.

### Key Features of Amp Code:

- **Sub-Agent Architecture:** Instead of using a single large model, Amp uses specialized sub-agents like the **Finder** (codebase search), the **Oracle** (deep reasoning/debugging), and the **Librarian** (fetching external framework context).

- **Dual Mode Operation:** The system features two primary agents: a **Smart Agent** (powered by Gemini 3 Flash for complex tasks) and a **Rush Agent** (optimized for speed and inner-loop edits).

- **Custom Terminal & Editor UI:** They built a dedicated terminal UI framework from scratch and a custom VS Code extension optimized for **code review** rather than manual editing.

- **Experimental Features:** The platform includes the **Kraken** agent for large-scale refactors via code mods and a unique "ad-sponsored" inference model to make the Rush agent free for side projects.

- **Collaboration Tools:** Includes the ability to **share threads** with teammates to exchange prompting techniques and debug agent workflows together.

### Strategic Takeaways:

Liu argues that as developers move toward becoming **reviewers** of agent-generated code, the focus must shift from "model selection" to building robust, agent-oriented architectures that can effectively navigate complex codebases.
