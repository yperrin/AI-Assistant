### **The 4 Levers to Customize Your AI**

1. **Memory:** Retaining personal context across conversations [05:15 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=315).

	- **ChatGPT:** Uses saved facts (explicit) and chat history (implicit). Use "Remember that..." to set reliable long-term preferences [05:36 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=336).

	- **Claude:** Focused on **Projects**. Each project has its own isolated memory space to prevent "context bleed" between different types of work [07:16 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=436).

	- **Gemini:** Leverages "Personal Intelligence" by connecting to your Google Workspace (Gmail, Calendar, Photos) [08:05 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=485).

1. **Instructions:** Persistent behavioral rules [08:41 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=521).

	- **The Key:** Avoid vague phrases like "be concise." Use specific logic: *"For factual questions, answer in one sentence; for analysis, walk through reasoning step-by-step"* [09:07 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=547).

	- **Claude Markdown (**`**.claudemd**`**):** For developers, treat this as a living document to encode project architecture and coding standards [10:09 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=609).

1. **Apps and Tools (MCP):** Capabilities like web search, code execution, and external integrations [10:47 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=647).

	- **Model Context Protocol (MCP):** An open standard (like "USB-C for AI") that allows models to connect to thousands of external servers [11:21 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=681).

	- **Intentionality:** Turning tools on/off changes the character of the model (e.g., preventing it from over-relying on web search when you want it to use internal knowledge) [13:12 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=792).

1. **Style and Tone Control:** Adjusting the "personality" of the output [13:23 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=803).

	- **ChatGPT:** Offers presets ranging from "friendly" to "cynical" with granular toggles for emojis and warmth [13:31 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=811).

	- **Claude:** Features a sophisticated "Custom Style" tool where you can upload samples of your own writing to have Claude mirror your voice and structure [14:11 Opens in a new window ](http://www.youtube.com/watch?v=KX0GurmgAoo&t=851).
