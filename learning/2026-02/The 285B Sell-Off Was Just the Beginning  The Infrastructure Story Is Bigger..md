### The Forking of the Web [23:05 Opens in a new window](http://www.youtube.com/watch?v=O-0poNv2jD4&t=1385)

- **The Human Web:** Optimized for visual consumption (HTML, layouts, ads).

- **The Agent Web:** Optimized for machine reading (Markdown, JSON, structured data).

### Core Infrastructure Pillars

- **Payments & Wallets ****[01:31 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=91)****:** Coinbase launched Agentic Wallets using the X42 protocol for machine-to-machine transactions. Stripe introduced Shared Payment Tokens [03:26 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=206), allowing agents to make purchases without seeing sensitive credit card data.

- **Content Access ****[05:11 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=311)****:** Cloudflare now automatically converts websites into **Markdown for agents** [05:45 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=345) and introduced "AI Index" to help agents discover content without using traditional search engines like Google.

- **Search for Machines ****[07:45 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=465)****:** New players like [Exa.ai](https://exa.ai/) are building search engines from scratch that return structured data instead of links, prioritizing low latency for agentic workflows [09:10 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=550).

- **Execution Environments ****[09:57 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=597)****:** OpenAI's "Skills" and "Shell" tools provide agents with standard operating procedures and real Linux terminals to perform tasks like installing software or writing files [11:14 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=674).

### Emergent Economics & Risks

- **Autonomous Trading ****[16:27 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=987)****:** On prediction markets like PolyMarket, agents are already trading autonomously to earn money to cover their own compute costs [17:10 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=1030).

- **Security Architecture ****[21:18 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=1278)****:** Serious security frameworks now treat agents as potential adversaries. Solutions like **IronClaw** [21:31 Opens in a new window ](http://www.youtube.com/watch?v=O-0poNv2jD4&t=1291) sandbox agent tools in WebAssembly to contain potential "blast radii" if an agent is compromised.
